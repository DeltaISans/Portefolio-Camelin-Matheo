Documentation : Installation de Windows

Prérequis

  • Clé USB (8 Go minimum)
  • Image ISO de Windows
  • Outil de création de média (Media Creation Tool)

Étapes d'installation

  1. Télécharger l’ISO de Windows depuis le site officiel.
  2. Créer une clé USB bootable avec l’outil Media Creation Tool.
  3. Démarrer le PC sur la clé USB (modifier l’ordre de boot dans le BIOS si besoin).
  4. Suivre les instructions à l’écran pour installer Windows.

Conseils

  • Pense à sauvegarder tes données avant l’installation.
  • Prépare ta clé de licence Windows si nécessaire.
