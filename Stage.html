Stage BTS SIO - Première Année

Automatisation du traitement des licences Microsoft 365

Entreprise : Groupe Mauvilac - Service Informatique

Période : 26 mai - 20 juin 2025

Durée : 4 semaines

Lieu : La Réunion, France

Fonction : Développeur - Équipe Helpdesk

Missions réalisées

  • Développement d'une solution d'automatisation pour le traitement des licences Microsoft 365 (1394+ utilisateurs répartis sur 12 sociétés du groupe)
  • Création de scripts Python pour l'analyse et la répartition automatique des licences par société avec gestion des cas complexes
  • Migration vers PowerShell pour une meilleure intégration Windows et utilisation dans des tâches planifiées
  • Conception d'une architecture modulaire avec fichiers de configuration JSON pour faciliter la maintenance
  • Documentation complète : guide d'installation, utilisation, dépannage et documentation technique

Compétences développées

  • Développement Python : Maîtrise d'openpyxl pour la manipulation Excel, gestion JSON, traitement de données complexes
  • Apprentissage PowerShell : COM Automation, Import-Csv, Export-Excel, scripting Windows avancé
  • Architecture logicielle : Conception modulaire, fichiers de configuration, versioning et maintenance
  • Méthodologie agile : Approche itérative, amélioration continue, tests et validation à chaque étape
  • Analyse et résolution de problèmes : Gestion des cas particuliers et optimisation

Résultats et bilan

Ce stage m'a permis de développer une solution concrète ayant un impact mesurable : réduction du temps de traitement de 2-3 heures à 2-3 minutes (98% de gain) avec élimination des erreurs humaines.

La satisfaction de voir l'outil utilisé quotidiennement par l'équipe et générant un gain de temps réel a été très motivante. Cette expérience a renforcé mon intérêt pour l'automatisation et l'optimisation des processus IT.
